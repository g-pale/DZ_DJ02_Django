# ScreenDjango - Учебный проект на Django

Учебный веб-проект на Django, демонстрирующий основы создания веб-приложений с использованием шаблонов, наследования и стилизации через Bootstrap.

## Описание проекта

ScreenDjango — это образовательный проект, созданный для изучения основ Django-разработки. Проект включает в себя 4 информационные страницы с единым дизайном и навигацией, демонстрируя ключевые концепции работы с шаблонами и статическими файлами.

## Основные возможности

- ✅ Использование базового шаблона с наследованием
- ✅ Модульная структура: отдельные файлы для меню и подвала
- ✅ Адаптивный дизайн на Bootstrap 5.3.3
- ✅ 4 информационные страницы с единым стилем
- ✅ Работа со статическими файлами (CSS, изображения)

## Технологии

- **Python** — язык программирования
- **Django 6.0** — веб-фреймворк
- **Bootstrap 5.3.3** — CSS-фреймворк для стилизации
- **SQLite** — база данных (по умолчанию)

## Установка и запуск

### Требования

- Python 3.8 или выше
- Django 6.0

### Шаги установки

1. **Клонируйте репозиторий или перейдите в директорию проекта:**
   ```bash
   cd DZ_DJ02_Django/DZ_DJ02
   ```

2. **Создайте виртуальное окружение (рекомендуется):**
   ```bash
   python -m venv venv
   source venv/bin/activate  # Для Linux/Mac
   # или
   venv\Scripts\activate  # Для Windows
   ```

3. **Установите зависимости:**
   ```bash
   pip install django
   ```

4. **Примените миграции (если необходимо):**
   ```bash
   python manage.py migrate
   ```

5. **Запустите сервер разработки:**
   ```bash
   python manage.py runserver
   ```

6. **Откройте браузер и перейдите по адресу:**
   ```
   http://127.0.0.1:8000/
   ```

## Особенности реализации

### Наследование шаблонов

Все страницы наследуют базовый шаблон `layoute.html` через директиву `{% extends %}`:

```django
{% extends 'main/layoute.html' %}

{% block content %}
    <!-- Уникальный контент страницы -->
{% endblock %}
```

### Модульная структура

Меню и подвал вынесены в отдельные файлы и подключаются через `{% include %}`:

- `nav.html` — навигационное меню с активными ссылками
- `footer.html` — подвал сайта с копирайтом

### Стилизация

- **Bootstrap 5.3.3** подключен через CDN в базовом шаблоне
- Пользовательские стили находятся в `static/main/css/main.css`
- Адаптивная верстка с использованием Bootstrap Grid System

## Маршрутизация

Маршруты определены в `main/urls.py`:

- `/` → `views.index` (name='home')
- `/page2` → `views.page2` (name='second')
- `/page3` → `views.page3` (name='third')
- `/page4` → `views.page4` (name='fourth')
